## 🚀 Featured Projects

### 🤖 Unitree Go2 – Intelligent Follow System

Designed and developed an autonomous follow system for the Unitree Go2 quadruped robot.

The system combines UWB-based positioning, YOLO object detection, and PID-based motion control to enable smooth, real-time human tracking.  
Built a modular behavior controller with multi-target locking and configurable control logic.

**Key Highlights:**
- Sensor fusion: UWB + Computer Vision  
- Real-time object detection using YOLO  
- Custom PID controller for motion stability  
- Modular architecture for behavior management  

**Tech Stack:**  
Python • Computer Vision • YOLO • PID Control • Robotics
